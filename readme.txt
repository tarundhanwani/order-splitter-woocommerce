Split an existing order of WooCommerce products easily into multiple orders.

== Description ==

This plugin lets you easily select products from an existing order in WooCommerce that has not been completed yet or cancelled. The new order that gets created from the split will have the same order number but with an additional text to indicate it is a clone or sub-order of the original.

== Features ==

* split any number of orders into multiple, sub-orders
* select as many or as little products from an existing non-complete/non-cancelled order
* the order number of the new order/sub-order will include the original order number with additional text to easily indicate it is a clone or a sub-order of the original.
* easy to use interface

== Localization ==

* english

== Installation ==

1. Install Order Splitter for WooCommerce by uploading the files to your server.
2. After activating the plugin, visit the Order section of WooCommerce to see the new Split button presented on each row of an order.
